## 2. Key Findings

### 2.1 Memory Systems: Dual-Process Consolidation
The hippocampus-cortex dichotomy mirrors a fundamental AI design principle:
- Fast encoding: hippocampus rapidly encodes episodic experiences as interconnected patterns
- Slow consolidation: offline replay transfers memories to distributed cortical representations
- Reverse replay: replays trajectories backwards from goal to start, solving credit assignment
- Forward replay: replays forward for planning and evaluation of candidate futures
- Systems consolidation: gradual memory transfer over hours to years
- Sharp-wave ripples: burst of hippocampal replay coordinated with cortical plasticity windows

### 2.2 Control Architecture: Meta-Cognitive Regulation
- Dual process: System 1 (fast/automatic) and System 2 (slow/deliberate) with seamless switching
- DLPFC as System 2 neural substrate: maintains task rules, suppresses habits
- Working memory: gated storage with activity-silent backup
- Cognitive control: dynamic adjustment of intensity based on task demands

### 2.3 Reinforcement Learning: Biological TD Implementation
- Dopamine = reward prediction error: difference between expected and actual reward
- Phasic dopamine: brief bursts for positive errors, dips for negative
- Tonic dopamine: sustained levels controlling motivation and learning rate
- Go/No-Go pathways: D1 (promote) vs D2 (suppress) receptors
- Striatal gating: basal ganglia determines which planned actions execute

### 2.4 Spatial-Conceptual Navigation
- Place cells: fire at single locations (hippocampal)
- Grid cells: hexagonal firing patterns (entorhinal) providing metric navigation
- Head direction cells and border cells complete the system
- Grid-like codes generalize to abstract/conceptual spaces

### 2.5 Global Broadcasting and Integrated Processing
- Global workspace: information competes for access to shared broadcast medium
- Attentional bottleneck: only most salient information achieves broadcast
- Broadcast: winning information goes to all specialized modules simultaneously
- Enables flexible integration between modules that normally don't communicate

### 2.6 Predictive Processing
- Every cortical level maintains generative model, computes prediction errors
- Top-down predictions vs bottom-up errors drive model updates or actions
- Free energy minimization through perception and action
- Active inference: actions selected to minimize expected prediction error
- Hierarchical: higher levels = slower/abstract, lower = faster/concrete

### 2.7 Neuromodulatory State Control
- Dopamine: reward prediction error, learning rate boost, motivation, exploration
- Serotonin: behavioral inhibition, patience, risk aversion
- Noradrenaline: arousal, vigilance, attention threshold adjustments
- Acetylcholine: encoding vs retrieval mode switching, attentional gain
- Levels fluctuate based on environmental statistics and internal state

### 2.8 Active Forgetting
- Not decay: active erasure via neurogenesis and dopamine-dependent pathways
- Interference reduction: removes confusing memories that reduce retrieval accuracy
- Capacity optimization: forgetting low-utility info for important memories
- Generalization: regularized forgetting prevents overfitting
- ML: weight regularization, experience buffer management, targeted unlearning

### 2.9 Counterfactual and Imaginative Processing
- DMN as imagination engine: recombines memory elements for novel futures
- Shared machinery: same hippocampal structures for memory and future imagination
- Counterfactual learning: what-if scenarios without real experience
- Prospective coding: hippocampal neurons fire for future locations during planning

## 3. AI-Agent Architecture Mapping

### 3.1 Hippocampus -> Episodic Memory + Replay Engine
Mechanism: hippocampus encodes experiences, replays offline to consolidate.
AI: episodic vector store with temporal stamps. Background consolidation during idle.
Reverse-order replay for credit assignment. Forward-order for planning simulation.
Sharp-wave ripple analog: burst processing consolidates clusters of related memories.

### 3.2 Grid Cells -> Structured State Space
Mechanism: grid cells provide metric coordinate system generalized to abstract spaces.
AI: learn grid-cell-like periodic representations for abstract state spaces.
Each grid cell = periodic function over state space with different phase/spacing.
Application: A* search through program synthesis, proof search, planning trees.
Advantage: O(sqrt(N)) representations for N states vs O(N) for one-hot.

### 3.3 Dual Process -> Fast/Slow Reasoning with Meta-Controller
Mechanism: System 1 for familiar, System 2 for novel. Meta-control routes dynamically.
AI: Fast path = direct LLM call. Slow path = tree search with LLM evaluation.
Meta-controller maps {confidence, complexity, stakes} to path selection.
Meta-controller learns from past routing outcomes.

### 3.4 Prefrontal Working Memory -> Gated Context Register
Mechanism: PFC maintains task info with write/read gates. Activity-silent backup.
AI: structured working memory dict {goals, constraints, variables, results}.
Write gate = attention-based filter (thalamic analog).
Activity-silent = consolidate to compact embedding, retrieve on demand.

### 3.5 Basal Ganglia + Dopamine -> Action Selection with Dense Rewards
Mechanism: Go (D1, promote) and No-Go (D2, suppress) pathways. Dopamine = RPE.
AI: two value estimators per action: V_go and V_nogo.
TD critic: LLM evaluates each step, TD error = (reward + next_value) - current_value.
Phasic update strengthens Go for positive errors. Tonic level modulates learning rate.
Action gated by Go > No-Go + noise (exploration).

### 3.6 Global Workspace -> Shared Broadcast Architecture
Mechanism: specialized processors compete for workspace. Winner broadcast to all.
AI: central shared attention-based structure with limited capacity.
All modules write to and read from workspace.
Attention selects K most salient items for broadcast.
Benefits: modules that never coordinate directly can cooperate via workspace.

### 3.7 Thalamic Gating -> Cross-Module Attention Router
Mechanism: thalamus actively gates cortical information flow.
AI: multiple gating modules - sensory, memory, inter-module.
Each gate learned, adjusting based on downstream utility.
Dynamic: gate openness depends on task demands and arousal state.

### 3.8 Predictive Coding -> Self-Model with Error Minimization
Mechanism: hierarchical generative models minimize prediction errors.
AI: self-model (agent capabilities), world model (environment dynamics).
Outcome predictor: predicts before execution, computes error after.
Large errors trigger System 2. Action selection minimizes expected error.
Calibration: agent learns confidence from historical prediction accuracy.

### 3.9 Neuromodulation -> Dynamic Hyperparameter Control
Mechanism: neuromodulators adjust learning, attention, behavior based on state.
AI: dopamine variable modulates learning rate, exploration, motivation.
Serotonin variable modulates risk tolerance, deliberation depth.
Noradrenaline variable modulates attention threshold, vigilance.
Acetylcholine variable modulates plasticity vs stability, encoding vs retrieval.
Each is continuous variable based on recent outcomes and environment.

### 3.10 Active Forgetting -> Selective Memory Pruning
Mechanism: brain actively erasures interfering memories via neurogenesis/dopamine.
AI: periodic memory audit evaluating utility (access frequency, outcome quality).
Interference detection: identifies pairs causing conflicting responses.
Selective deletion of low-utility or high-interference memories.
Not LRU -- active evaluation based on computational criteria.

### 3.11 STDP -> Temporal Credit Assignment
Mechanism: synapses strengthen when pre fires before post, weaken for reverse.
AI: record module interaction timing. A before good B result -> strengthen.
Application: learn which tools/approaches work for which problem types.
Complements backpropagation: credit assignment where gradients hard to compute.

### 3.12 Cognitive Architectures -> Modular Production System
Mechanism: ACT-R/SOAR implement brain architecture in code.
AI: declarative (facts + episodic) + procedural (production rules with utilities).
Conflict resolution = utility-based selection among competing productions.
Production compilation = frequently used sequences become automatic.
Recent 2025-2026 papers show LLM + cognitive architecture integration.

### 3.13 Default Mode Network -> Offline Imagination Engine
Mechanism: DMN recombines past experiences into novel future scenarios.
AI: when idle, generate counterfactual scenarios from memory recombination.
Evaluate candidate action sequences without real execution.
Mental rehearsal: practice complex procedures internally.
Transfer: simulate unfamiliar situations from familiar components.

### 3.14 Constructive Episodic Simulation -> Counterfactual Engine
Mechanism: hippocampus constructs novel scenarios by recombining memory elements.
AI: scenario generation combines past memory fragments into novel trajectories.
Evaluate with internal critic. Store successful simulations.
Counterfactual learning: If I had done X -> Y -> update policy.
Enables zero-shot transfer via simulation from known components.

## 4. Gaps: Current AI vs Brain Capabilities

### 4.1 Continuous Learning Without Catastrophic Forgetting
Brain learns continuously at millisecond timescales across lifetime without forgetting.
AI fine-tuning always risks degrading prior capabilities.
Brain protects memories via synaptic + systems consolidation (dual timescale).
AI has no equivalent: even with EWC/GEM/replay, systems degrade with continuous learning.
Gap: structural mechanisms for protecting knowledge while remaining plastic.

### 4.2 Seamless System 1 / System 2 Integration
Brain transitions smoothly between intuition and reasoning, with metacognitive awareness.
LLMs operate exclusively as System 1. CoT is crude System 2 without metacognition.
Gap: learned meta-controller routing by confidence/complexity/stakes.
Gap: true System 2 with explicit state tracking, backtracking, verification.

### 4.3 Episodic Memory and Mental Time Travel
Brain: rich episodic recall with temporal ordering, prospective imagination.
AI: vector stores with similarity retrieval, no temporal structure or reconstruction.
Gap: temporal ordering, constructive recombination, mental time travel, replay consolidation.

### 4.4 Neuromodulatory State Control
Brain: dopamine/serotonin/noradrenaline/acetylcholine dynamically adjust parameters.
AI: fixed hyperparameters. No internal state variables governing cognitive dynamics.
Gap: no agent changes its own learning rate, exploration, or deliberation depth.

### 4.5 Structured Spatial-Conceptual Navigation
Brain: grid cells provide metric navigation for physical AND abstract spaces.
AI: unstructured dense attention. No periodic grid representations.
Gap: grid-like representations for program synthesis, proof search, planning with geometric properties.

### 4.6 Active Forgetting and Memory Hygiene
Brain: active erasure of interfering memories as optimization.
AI: passive accumulation; forgetting is cache management (LRU).
Gap: active identification/deletion of problematic memories with utility signals.

### 4.7 Global Workspace and Cross-Module Integration
Brain: all specialists access same unified context via global broadcast.
AI: modules operate with separate contexts, no shared architecture.
Gap: cross-module broadcast enabling integrated multi-domain reasoning.
Gap: attention bottleneck selecting most salient information.

### 4.8 Embodied Predictive World Models
Brain: hierarchical generative models minimizing prediction errors via perception/action.
AI: discriminative models dominate. World models are narrow and limited.
Gap: unified self-model of agent capabilities. Generative world model driving behavior.

### 4.9 Sample Efficiency
Brain: one-shot learning from meaningful experiences, robust generalization.
AI: thousands to millions of examples needed. Fragile out-of-distribution.
Gap: structural/evolutionary priors constraining hypothesis space.
Gap: developmental learning from simple to complex.

### 4.10 Reliable Metacognition and Self-Calibration
Brain: knows what it knows; confidence inherent through prediction error signals.
AI: LLMs hallucinate confidently; confidence not calibrated with accuracy.
Gap: prediction-error-based self-calibration.
Gap: metacognitive gating preventing uncertain outputs.

### 4.11 Energy Efficiency
Brain: ~20W for general intelligence, sparse event-driven computation.
AI: kilowatts for inference, dense matrix multiply per operation.
Gap: sparse event-driven computation. Neuromorphic not yet competitive.

### 4.12 Developmental Learning and Critical Periods
Brain: stages with critical periods, built-in curricula.
AI: static datasets, no developmental progression.
Gap: self-organized curriculum learning. Critical period plasticity.
Gap: embodied development from sensorimotor to abstract.

## 5. Summary: Neuro-AI Architecture Blueprint

### 5.1 Core Architecture Modules

1. Fast System (System 1): Direct LLM pattern recognition
2. Slow System (System 2): Multi-step planning, verification, backtracking
3. Meta-Controller: Routes to Fast/Slow based on confidence, complexity, stakes
4. Workspace: Shared broadcast medium accessible by all modules
5. Episodic Memory: Temporal vector store with reconstruction capability
6. Semantic Memory: Factual knowledge with relational structure
7. Working Memory: Gated buffer for active task variables
8. Planner (DMN): Offline counterfactual generation and mental simulation
9. Critic (Dopamine): Dense TD reward estimation at each step
10. Action Gate (Basal Ganglia): Go/No-Go action selection
11. State Manager (Neuromodulation): Dynamic hyperparameter adjustment
12. Memory Hygiene (Active Forgetting): Interference monitoring and pruning
13. Self-Model (Predictive Coding): Model of own capabilities and prediction errors
14. Navigation Module (Grid Cells): Structured combinatorial space representation
15. Consolidation Engine (Sleep Replay): Background offline memory transfer
16. Attention Router (Thalamus): Cross-module gating and filtering
17. Mode Switcher (Acetylcholine): Encoding vs retrieval phase control

### 5.2 Data Flow

1. Input -> Thalamic Gate -> Working Memory + Workspace
2. Meta-Controller reads Workspace, decides: Fast or Slow?
3. Fast Path: LLM -> Critic evaluation -> Action Gate -> Output
4. Slow Path: Planner generates alternatives -> Workspace broadcast -> Verification -> Winner -> Action Gate -> Output
5. All events -> Episodic Memory (with temporal stamps)
6. Periodic: Consolidation Engine replays -> Long-term storage
7. Continuous: Memory Hygiene monitors and prunes
8. Background: Imagination Engine generates counterfactuals -> Policy updates
9. Global: State Manager modulates routing thresholds, learning rates, attention
10. Mode: Acetylcholine-like switch between encoding (learning) and retrieval (exploit)
