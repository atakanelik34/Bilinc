## KEY ECOSYSTEM INSIGHTS

1. **Memory management is exploding** — 30+ dedicated memory packages on PyPI, up from almost none 2 years ago
2. **Vector DBs are commoditized** — 13 packages, all doing similar things. Differentiation is in integration patterns
3. **Graph + Vector is the winning combo** — graph-vect-rag, neo4j-graphrag, graphrag all combine both
4. **Brain-inspired packages are emerging** — braincog, neural-memory, neuro-cortex-memory, cerebro-memory
5. **Forgetting is still rare** — Only memory-decay and engram-memory implement forgetting curves
6. **TypeScript is catching up** — Many GitHub projects are TS-first, Python still dominates PyPI
7. **LangChain ecosystem dominates** — langmem, langchain-mengram, neuromem-sdk all integrate with LangChain
8. **MCP (Model Context Protocol) is the new standard** — neural-memory (56 MCP tools), neuro-cortex-memory, engram-memory all support MCP
