## 1. 1. Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers (2026)
- URL: https://arxiv.org/abs/2603.07670v1
- Summary: Comprehensive survey covering memory mechanisms for autonomous LLM agents, evaluation methodologies, and emerging frontiers in agent memory design. Covers short-term, long-term, and working memory paradigms.
- Relevance: Foundational survey that maps the entire landscape of memory mechanisms needed for autonomous agents.

## 2. 2. Memory in the Age of AI Agents (2025)
- URL: https://arxiv.org/abs/2512.13564
- Summary: 102-page survey on memory systems in the age of AI agents. Covers memory architectures, retrieval strategies, consolidation mechanisms, and evaluation benchmarks for agent memory.
- Relevance: Essential reference for understanding the full spectrum of memory types an AI agent brain needs.

## 3. 3. AI Meets Brain: A Unified Survey on Memory Systems from Cognitive Neuroscience to Autonomous Agents (2025)
- URL: https://arxiv.org/abs/2512.23343
- Summary: Bridges cognitive neuroscience memory models with AI agent memory systems. Proposes unified framework mapping sensory, working, episodic, and semantic memory to computational architectures.
- Relevance: Directly maps biological brain memory systems to AI agent architecture design.

## 4. 4. Memory OS of AI Agent (2025)
- URL: https://aclanthology.org/2025.emnlp-main.1318.pdf
- Summary: Proposes a Memory Operating System for AI agents that manages memory lifecycle including encoding, storage, retrieval, consolidation, and forgetting. Published at EMNLP 2025.
- Relevance: Provides OS-level abstraction for managing all memory operations in an agent brain.

## 5. 5. A-Mem: Agentic Memory for LLM Agents (2025)
- URL: https://arxiv.org/abs/2502.12110
- Summary: Introduces A-MEM, an agentic memory system that enables LLM agents to actively manage their own memory through reading, writing, and updating operations. NeurIPS 2025.
- Relevance: Shows how agents can be given agency over their own memory management.

## 6. 6. General Agentic Memory Via Deep Research (2025)
- URL: https://arxiv.org/abs/2511.18423
- Summary: Proposes using deep research processes to build general agentic memory, enabling agents to accumulate and leverage knowledge across diverse tasks through structured research workflows.
- Relevance: Demonstrates how research-driven memory accumulation can power general-purpose agents.

## 7. 7. Mem0: Building Production-Ready AI Agents with Scalable Long-Term Memory (2025)
- URL: https://arxiv.org/abs/2504.19413
- Summary: Presents Mem0, a production-ready memory layer for AI agents that provides scalable long-term memory with extraction, consolidation, and retrieval capabilities.
- Relevance: Practical, deployable memory system for production AI agent brains.

## 8. 8. AtomMem: Learnable Dynamic Agentic Memory with Atomic Memory Operation (2026)
- URL: https://arxiv.org/abs/2601.08323
- Summary: Introduces AtomMem, a learnable dynamic memory system using atomic memory operations for fine-grained control over agent memory read/write/update/delete.
- Relevance: Atomic operations provide precise memory control needed for a sophisticated agent brain.

## 9. 9. Memp: Exploring Agent Procedural Memory (2025)
- URL: https://arxiv.org/abs/2508.06433
- Summary: Explores procedural memory for AI agents — memory for skills, habits, and learned procedures. Proposes framework for storing and retrieving action sequences.
- Relevance: Procedural memory is essential for agent skill acquisition and habit formation.

## 10. 10. Remember Me, Refine Me: A Dynamic Procedural Memory Framework for Experience-Driven Agent Evolution (2025)
- URL: https://arxiv.org/abs/2512.10696
- Summary: Dynamic procedural memory framework that enables agents to evolve through experience, refining stored procedures based on outcomes and feedback.
- Relevance: Enables the agent brain to improve its procedural knowledge over time.

## 11. 11. Agentic Memory: Learning Unified Long-Term and Short-Term Memory Management for LLM Agents (2026)
- URL: https://arxiv.org/abs/2601.01885
- Summary: Proposes learned unified memory management that jointly optimizes long-term and short-term memory operations for LLM agents through reinforcement learning.
- Relevance: Unified memory management is core to an efficient agent brain architecture.

## 12. 12. All-Mem: Agentic Lifelong Memory via Dynamic Topology Evolution (2026)
- URL: https://arxiv.org/abs/2603.19595
- Summary: Introduces All-Mem, a lifelong memory system with dynamically evolving topology that adapts its structure based on accumulated experience and knowledge patterns.
- Relevance: Dynamic topology evolution enables the agent brain to reorganize as it learns.

## 13. 13. Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory (2026)
- URL: https://arxiv.org/abs/2603.04257
- Summary: Proposes indexed experience memory with RL-based retrieval for scaling long-horizon LLM agents, enabling efficient recall of relevant past experiences.
- Relevance: Experience indexing is critical for long-horizon agent tasks.

## 14. 14. AriadneMem: Threading the Maze of Lifelong Memory for LLM Agents (2026)
- URL: https://arxiv.org/abs/2603.03290
- Summary: Proposes AriadneMem, a lifelong memory system that threads together disparate experiences into coherent memory structures for LLM agents operating over extended periods.
- Relevance: Coherent memory threading prevents fragmentation in long-running agents.

## 15. 15. Continuum Memory Architectures for Long-Horizon LLM Agents (2026)
- URL: https://arxiv.org/abs/2601.09913
- Summary: Introduces continuum memory architectures that provide seamless memory access across arbitrary time horizons, addressing the temporal scaling challenge in LLM agents.
- Relevance: Enables the agent brain to operate effectively across any timescale.

## 16. 16. Membox: Weaving Topic Continuity into Long-Range Memory for LLM Agents (2026)
- URL: https://arxiv.org/abs/2601.03785
- Summary: Weaves topic continuity signals into long-range memory, enabling agents to maintain thematic coherence across extended interactions and tasks.
- Relevance: Topic continuity is essential for coherent long-term agent behavior.

## 17. 17. ByteRover: Agent-Native Memory Through LLM-Curated Hierarchical Context (2026)
- URL: https://arxiv.org/abs/2604.01599
- Summary: Proposes LLM-curated hierarchical context management for agent-native memory, enabling multi-level abstraction in memory organization and retrieval.
- Relevance: Hierarchical memory organization mirrors brain's cortical hierarchy.
