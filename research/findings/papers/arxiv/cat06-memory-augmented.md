## 1. 60. Memory-Augmented Transformers: A Systematic Review from Neuroscience Principles to Enhanced Model Architectures (2025)
- URL: https://arxiv.org/abs/2508.10824
- Summary: Systematic review bridging neuroscience memory principles with memory-augmented transformer architectures, covering external memory, attention-based memory, and recurrent memory mechanisms.
- Relevance: Neuroscience-grounded review directly informs agent brain memory architecture.

## 2. 61. MATTER: Memory-Augmented Transformer Using Heterogeneous Knowledge Sources (2024)
- URL: https://arxiv.org/abs/2406.04670
- Summary: Introduces MATTER, a memory-augmented transformer that integrates heterogeneous knowledge sources (text, graphs, databases) into unified memory for enhanced generation.
- Relevance: Heterogeneous knowledge integration is essential for a multi-modal agent brain.

## 3. 62. TransformerFAM: Feedback Attention is Working Memory (2024)
- URL: https://arxiv.org/abs/2404.09173
- Summary: Proposes that feedback attention mechanisms in transformers serve as working memory, providing theoretical analysis and architectural improvements for enhanced working memory.
- Relevance: Establishes transformer attention as working memory mechanism for the agent brain.

## 4. 63. An Evolved Universal Transformer Memory (2024)
- URL: https://arxiv.org/abs/2410.13166
- Summary: Proposes evolved universal transformer memory that adapts its memory mechanism through evolutionary strategies, reducing costs while maintaining performance.
- Relevance: Evolved memory mechanisms provide efficient alternatives for the agent brain.

## 5. 64. MELODI: Exploring Memory Compression for Long Contexts (2024)
- URL: https://arxiv.org/abs/2410.03156
- Summary: Presents MELODI, a novel memory compression architecture for long contexts in transformers, enabling efficient storage and retrieval of extended sequences. Google DeepMind.
- Relevance: Memory compression enables the agent brain to handle vast context efficiently.

## 6. 65. Stateful Memory-Augmented Transformers for Efficient Dialogue Modeling (2024)
- URL: https://aclanthology.org/2024.findings-eacl.57.pdf
- Summary: Introduces stateful memory-augmented transformers for dialogue, maintaining persistent memory states across turns for coherent multi-turn conversations. EACL 2024.
- Relevance: Stateful memory enables persistent agent brain state across interactions.

## 7. 66. Memory-augmented Transformers can implement Linear First-Order Optimization Methods (2024)
- URL: https://arxiv.org/abs/2410.07263
- Summary: Theoretical analysis showing that memory-augmented transformers can implement linear first-order optimization methods, bridging optimization theory with transformer memory.
- Relevance: Theoretical foundation for optimization-based memory in the agent brain.

## 8. 67. Adaptive Loops and Memory in Transformers: Think Harder or Know More? (2026)
- URL: https://arxiv.org/abs/2603.08391
- Summary: Explores trade-off between adaptive computation loops (thinking harder) and memory augmentation (knowing more) in transformers, providing guidelines for architecture design.
- Relevance: Critical trade-off analysis for agent brain compute vs. memory allocation.
