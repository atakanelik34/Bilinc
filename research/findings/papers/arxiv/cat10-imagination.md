## 1. 98. Imagine-then-Plan: Agent Learning from Adaptive Lookahead with World Models (2026)
- URL: https://arxiv.org/abs/2601.08955
- Summary: Proposes imagine-then-plan paradigm where agents first imagine possible futures using world models, then plan optimal actions based on simulated outcomes.
- Relevance: Imagination-driven planning enables the agent brain to simulate before acting.

## 2. 99. Dyna-Mind: Learning to Simulate from Experience for Better AI Agents (2025)
- URL: https://arxiv.org/abs/2510.09577
- Summary: Proposes Dyna-Mind framework where agents learn to simulate environments from experience, enabling model-based planning without explicit environment access. Under review at ICLR 2026.
- Relevance: Experience-based simulation enables the agent brain to mentally rehearse actions.

## 3. 100. SimuRA: A World-Model-Driven Simulative Reasoning Architecture for General Goal-Oriented Agents (2025)
- URL: https://arxiv.org/abs/2507.23773
- Summary: Proposes SimuRA, a simulative reasoning architecture driven by LLM-based world models, enabling general goal-oriented agents to reason through mental simulation.
- Relevance: Simulative reasoning architecture is the imagination engine of the agent brain.

## 4. 101. "Just in Time" World Modeling Supports Human Planning and Reasoning (2026)
- URL: https://arxiv.org/abs/2601.14514
- Summary: Shows that humans construct world models "just in time" for specific planning tasks, proposing computational framework for on-demand world model construction in AI.
- Relevance: On-demand world modeling enables efficient agent brain simulation.

## 5. 102. WorldCoder, a Model-Based LLM Agent: Building World Models by Writing Code (2024)
- URL: https://arxiv.org/abs/2405.14205
- Summary: Model-based LLM agent that builds world models by writing and executing code, enabling it to interact with and learn about its environment programmatically. NeurIPS 2024.
- Relevance: Code-based world model building gives the agent brain executable mental models.

## 6. 103. Agent Planning with World Knowledge Model (2024)
- URL: https://arxiv.org/abs/2405.14205
- Summary: Proposes using world knowledge models for agent planning, enabling agents to leverage structured world knowledge for more effective task planning and execution.
- Relevance: World knowledge models provide the factual basis for agent brain imagination.

## 7. 104. Is Your LLM Secretly a World Model of the Internet? Model-Based Planning for Web Agents (2024)
- URL: https://arxiv.org/abs/2411.06559
- Summary: Investigates whether LLMs implicitly encode world models of the internet, and leverages this for model-based planning in web navigation tasks.
- Relevance: Implicit world models in LLMs can be exploited for agent brain planning.

## 8. 105. World Modelling Improves Language Model Agents (2025)
- URL: https://arxiv.org/abs/2506.02918
- Summary: Demonstrates that explicit world modeling significantly improves language model agent performance across diverse tasks, providing empirical evidence for model-based approaches.
- Relevance: Empirical validation that world models improve agent brain performance.

## 9. 106. Can Mental Imagery Improve the Thinking Capabilities of AI Systems? (2025)
- URL: https://arxiv.org/abs/2507.12555
- Summary: Investigates whether mental imagery mechanisms can improve AI system thinking capabilities, drawing parallels between human mental imagery and AI visualization.
- Relevance: Mental imagery adds visual-spatial reasoning to the agent brain.

## 10. 107. SpatialDreamer: Incentivizing Spatial Reasoning via Active Mental Imagery (2025)
- URL: https://arxiv.org/abs/2507.12556
- Summary: Proposes active mental imagery framework for incentivizing spatial reasoning in AI systems, enabling agents to construct and manipulate mental spatial representations.
- Relevance: Spatial reasoning through mental imagery enhances agent brain's environmental understanding.
