## 1. 18. Chain-of-Thought Prompting Elicits Reasoning in Large Language Models (2022)
- URL: https://arxiv.org/abs/2201.11903
- Summary: Seminal paper introducing Chain-of-Thought prompting, showing that LLMs can perform complex reasoning by generating intermediate reasoning steps. Demonstrates dramatic improvements on math and logic tasks.
- Relevance: Foundational reasoning technique — the agent brain needs step-by-step reasoning as a core capability.

## 2. 19. ReAct: Synergizing Reasoning and Acting in Language Models (2022)
- URL: https://arxiv.org/abs/2210.03629
- Summary: Introduces ReAct framework that interleaves reasoning traces and task-specific actions, enabling LLMs to both reason about and act in environments. Combines thought generation with action execution.
- Relevance: Core paradigm for agent brain — reasoning and acting must be tightly integrated.

## 3. 20. Graph of Thoughts: Solving Elaborate Problems with Large Language Models (2023)
- URL: https://arxiv.org/abs/2308.09687
- Summary: Introduces Graph of Thoughts (GoT), modeling LLM reasoning as arbitrary graph structures where thoughts can be aggregated, transformed, and combined in non-linear ways. Published at AAAI 2024.
- Relevance: Graph-based reasoning enables the agent brain to handle complex, interconnected thought processes.

## 4. 21. Demystifying Chains, Trees, and Graphs of Thoughts (2024)
- URL: https://arxiv.org/abs/2401.14295
- Summary: Systematic analysis unifying Chain-of-Thought, Tree-of-Thoughts, and Graph-of-Thoughts frameworks. Provides theoretical foundations and empirical comparisons across reasoning structures.
- Relevance: Unified framework for selecting the right reasoning structure for the agent brain.

## 5. 22. Tree-of-Thoughts: Deliberate Problem Solving with Large Language Models (2023)
- URL: https://arxiv.org/abs/2305.10601
- Summary: Introduces Tree-of-Thoughts framework that explores multiple reasoning paths in a tree structure, enabling deliberate search, lookahead, and backtracking for complex problem solving.
- Relevance: Tree search enables the agent brain to explore multiple futures before acting.

## 6. 23. Forest-of-Thought: Scaling Test-Time Compute for Enhancing LLM Reasoning (2024)
- URL: https://arxiv.org/abs/2412.09078
- Summary: Extends Tree-of-Thoughts to Forest-of-Thoughts, scaling test-time compute by running multiple reasoning trees in parallel and aggregating results for enhanced reasoning.
- Relevance: Parallel reasoning forests enable the agent brain to scale computation for hard problems.

## 7. 24. Adaptive Graph of Thoughts: Test-Time Adaptive Reasoning Unifying Chain, Tree, and Graph Structures (2025)
- URL: https://arxiv.org/abs/2502.05078
- Summary: Proposes adaptive reasoning that dynamically selects between chain, tree, and graph structures at test time based on problem complexity and characteristics.
- Relevance: Adaptive reasoning structure selection optimizes agent brain compute allocation.

## 8. 25. Framework of Thoughts: A Foundation Framework for Dynamic and Optimized Reasoning (2026)
- URL: https://arxiv.org/abs/2602.16512
- Summary: Proposes a foundation framework that dynamically optimizes reasoning across chains, trees, and graphs, providing a unified approach to LLM reasoning structure selection.
- Relevance: Dynamic reasoning optimization is key to an efficient agent brain.

## 9. 26. On the Diagram of Thought (2024)
- URL: https://arxiv.org/abs/2409.10038
- Summary: Introduces Diagram of Thought, a reasoning framework that uses visual-spatial diagrammatic representations to enhance LLM reasoning capabilities beyond text-only approaches.
- Relevance: Diagrammatic reasoning adds spatial visualization to the agent brain.

## 10. 27. Hierarchical Chain-of-Thought Prompting: Enhancing LLM Reasoning Performance and Efficiency (2026)
- URL: https://arxiv.org/abs/2604.00130
- Summary: Proposes hierarchical CoT that organizes reasoning at multiple levels of abstraction, improving both performance and efficiency on complex reasoning tasks.
- Relevance: Hierarchical reasoning mirrors the brain's multi-level cognitive processing.

## 11. 28. MTMT: Consolidating Multiple Thinking Modes to Form a Thought Tree (2024)
- URL: https://arxiv.org/abs/2412.03987
- Summary: Consolidates multiple thinking modes (analytical, creative, systematic) into unified thought trees, strengthening LLM reasoning through mode diversity.
- Relevance: Multi-mode thinking enriches the agent brain's cognitive diversity.

## 12. 29. Chain of Preference Optimization: Improving Chain-of-Thought Reasoning in LLMs (2024)
- URL: https://arxiv.org/abs/2406.09171
- Summary: Introduces Chain of Preference Optimization (CPO) for improving CoT reasoning through preference-based training on reasoning trajectories. NeurIPS 2024.
- Relevance: Preference-optimized reasoning improves the quality of agent brain thought processes.

## 13. 30. Pre-Act: Multi-Step Planning and Reasoning Improves Acting in LLM Agents (2025)
- URL: https://arxiv.org/abs/2505.09970
- Summary: Shows that multi-step planning before acting significantly improves LLM agent performance, proposing Pre-Act framework for deliberative planning.
- Relevance: Pre-action planning is essential for deliberate agent brain behavior.

## 14. 31. Beyond ReAct: A Planner-Centric Framework for Complex Tool-Augmented LLM Reasoning (2025)
- URL: https://arxiv.org/abs/2511.10037
- Summary: Proposes planner-centric framework that goes beyond ReAct by separating planning from execution, enabling more sophisticated tool-augmented reasoning.
- Relevance: Planner-centric architecture separates strategic from tactical thinking in the agent brain.

## 15. 32. IMAGINE: Integrating Multi-Agent System into One Model for Complex Reasoning and Planning (2025)
- URL: https://arxiv.org/abs/2510.14406
- Summary: Integrates multi-agent reasoning patterns into a single model, enabling complex reasoning and planning through internal agent simulation.
- Relevance: Internal multi-agent simulation enables the agent brain to self-debate.
