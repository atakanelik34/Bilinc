## 1. 89. Continual Learning and Catastrophic Forgetting (2024)
- URL: https://arxiv.org/abs/2403.05175
- Summary: Comprehensive review of catastrophic forgetting in continual learning, covering mechanisms, mitigation strategies, and biological parallels to human memory consolidation.
- Relevance: Understanding catastrophic forgetting is essential for building an agent brain that retains knowledge.

## 2. 90. Catastrophic Forgetting in LLMs: A Comparative Analysis Across Language Tasks (2025)
- URL: https://arxiv.org/abs/2504.01241
- Summary: Comparative analysis of catastrophic forgetting across diverse language tasks in LLMs, identifying which types of knowledge are most vulnerable to forgetting.
- Relevance: Task-specific forgetting analysis informs which agent brain memories need protection.

## 3. 91. Mitigating Catastrophic Forgetting in Continual Learning through Model Growth (2025)
- URL: https://arxiv.org/abs/2509.01213
- Summary: Proposes model growth as a strategy to mitigate catastrophic forgetting, dynamically expanding model capacity as new tasks are learned rather than overwriting existing knowledge.
- Relevance: Model growth enables the agent brain to expand without forgetting.

## 4. 92. FIT: Defying Catastrophic Forgetting in Continual LLM Unlearning (2026)
- URL: https://arxiv.org/abs/2601.21682
- Summary: Introduces FIT framework that defies catastrophic forgetting during continual LLM unlearning, enabling selective knowledge removal while preserving unrelated capabilities.
- Relevance: Selective unlearning enables the agent brain to forget harmful information selectively.

## 5. 93. Evolutionary Strategies Lead to Catastrophic Forgetting in LLMs (2026)
- URL: https://arxiv.org/abs/2601.20861
- Summary: Shows that evolutionary strategies for LLM training can lead to catastrophic forgetting, analyzing the mechanisms and proposing mitigation approaches.
- Relevance: Warning about evolutionary training's forgetting risks for the agent brain.

## 6. 94. Reinforcement Fine-Tuning Naturally Mitigates Forgetting in Continual Post-Training (2025)
- URL: https://arxiv.org/abs/2507.05386
- Summary: Shows that RL-based fine-tuning naturally mitigates catastrophic forgetting during continual post-training, providing theoretical and empirical analysis.
- Relevance: RL fine-tuning provides a natural forgetting mitigation mechanism for the agent brain.

## 7. 95. Dynamic Orthogonal Continual Fine-tuning for Mitigating Catastrophic Forgettings (2025)
- URL: https://arxiv.org/abs/2509.23893
- Summary: Proposes dynamic orthogonal subspaces for continual fine-tuning, ensuring new learning occupies orthogonal parameter space to prevent interference with existing knowledge.
- Relevance: Orthogonal learning enables the agent brain to learn without interference.

## 8. 96. Unlocking the Power of Function Vectors for Characterizing and Mitigating Catastrophic Forgetting (2025)
- URL: https://arxiv.org/abs/2502.11019
- Summary: Uses function vectors to characterize and mitigate catastrophic forgetting in continual instruction tuning, providing interpretable representations of knowledge changes. ICLR 2025.
- Relevance: Function vectors provide interpretable knowledge tracking for the agent brain.

## 9. 97. Continual Learning in Large Language Models: Methods, Challenges, and Opportunities (2026)
- URL: https://arxiv.org/abs/2603.12658
- Summary: Comprehensive survey of continual learning methods for LLMs, covering approaches to mitigate forgetting, evaluation benchmarks, and open research challenges.
- Relevance: Definitive survey on continual learning for the agent brain's lifelong learning capability.
