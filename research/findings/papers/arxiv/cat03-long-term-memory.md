## 1. 33. Augmenting Language Models with Long-Term Memory (2023)
- URL: https://arxiv.org/abs/2306.07174
- Summary: Proposes augmenting LLMs with explicit long-term memory through a memory module that can be read from and written to, enabling persistent knowledge storage beyond context windows.
- Relevance: Foundational work on adding persistent memory to language models.

## 2. 34. M+: Extending MemoryLLM with Scalable Long-Term Memory (2025)
- URL: https://openreview.net/forum?id=OcqbkROe8J
- Summary: Extends MemoryLLM with scalable long-term memory capabilities, ICML 2025. Proposes efficient memory scaling mechanisms for handling growing knowledge bases.
- Relevance: Scalable memory is essential for agents that accumulate knowledge over time.

## 3. 35. Long Term Memory: The Foundation of AI Self-Evolution (2024)
- URL: https://arxiv.org/abs/2410.15665
- Summary: Argues that long-term memory is the foundation for AI self-evolution, proposing frameworks for memory-driven continuous improvement in AI systems.
- Relevance: Establishes long-term memory as prerequisite for agent self-improvement.

## 4. 36. Human-Inspired Episodic Memory for Infinite Context LLMs (2025)
- URL: https://arxiv.org/abs/2407.09450
- Summary: Proposes human-inspired episodic memory system enabling LLMs to handle infinite context through selective encoding, consolidation, and retrieval of experiences. ICLR 2025.
- Relevance: Human-inspired episodic memory directly informs agent brain design.

## 5. 37. Preference-Aware Memory Update for Long-Term LLM Agents (2025)
- URL: https://arxiv.org/abs/2510.09720
- Summary: Introduces preference-aware memory update mechanisms that prioritize storing information aligned with user preferences and agent goals in long-term memory.
- Relevance: Preference-aware memory ensures the agent brain stores what matters most.

## 6. 38. Beyond a Million Tokens: Benchmarking and Enhancing Long-Term Memory in LLMs (2026)
- URL: https://arxiv.org/abs/2510.27246
- Summary: ICLR 2026 paper benchmarking long-term memory capabilities beyond million-token contexts, proposing enhancements for ultra-long-context memory in LLMs.
- Relevance: Benchmarks and enhances the agent brain's capacity for vast memory.

## 7. 39. Memory in Large Language Models: Mechanisms, Evaluation and Evolution (2025)
- URL: https://arxiv.org/abs/2509.18868
- Summary: Comprehensive survey under unified operationalized definition of memory in LLMs, covering mechanisms, evaluation methods, and evolutionary trajectories.
- Relevance: Unified framework for understanding all memory mechanisms in the agent brain.

## 8. 40. MLP Memory: A Retriever-Pretrained Memory for Large Language Models (2025)
- URL: https://arxiv.org/abs/2508.01832
- Summary: Proposes MLP-based memory module pretrained as a retriever, enabling efficient memory access within LLM architectures without external retrieval systems.
- Relevance: Internal retriever-pretrained memory reduces agent brain latency.

## 9. 41. Memory in the LLM Era: Modular Architectures and Strategies in a Unified Framework (2026)
- URL: https://arxiv.org/abs/2604.01707
- Summary: Proposes modular memory architectures with unified benchmarking, comparing different memory strategies and their effectiveness across tasks.
- Relevance: Modular approach enables flexible agent brain memory composition.

## 10. 42. Larimar: Large Language Models with Episodic Memory Control (2024)
- URL: https://arxiv.org/abs/2403.11901
- Summary: Introduces Larimar, giving LLMs explicit episodic memory control mechanisms for storing, retrieving, and utilizing specific past experiences during generation.
- Relevance: Episodic memory control enables the agent brain to recall specific events.
