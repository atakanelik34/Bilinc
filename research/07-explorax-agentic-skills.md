These represent the PROCEDURAL MEMORY layer — what agents can DO, not just know.

## KEY INSIGHT
Procedural memory IS skills. As agents gain experience, skill library should:
1. Grow (learn new skills)
2. Prune (forget unused)
3. Abstract (combine into meta-skills)
4. Transfer (apply across domains)
